Dieses Skript lädt die kostenlos verfügbaren Känguru-Comics von der Website “https://www.zeit.de/serie/die-kaenguru-comics” herunter und speichert sie in einem vom Benutzer angegebenen Ordner auf seinem lokalen Computer. 
Der Benutzer wird aufgefordert, den Pfad auszuwählen, an dem der Ordner “K-Comics” erstellt werden soll, und das Skript erstellt den Ordner am ausgewählten Pfad, wenn er noch nicht existiert. 
Das Skript lädt dann alle verfügbaren Känguru-Comics von der Website herunter und speichert jedes Bild als einzelne Datei mit absteigenden Zahlen, beginnend bei 627 (letzte Ausgabe). Schließlich kombiniert das Skript alle heruntergeladenen Bilder in einer einzigen PDF-Datei und speichert sie im selben Ordner.
